# Machine Learning Project: Bank Transactions Analysis

## Deskripsi
Proyek ini berfokus pada analisis data transaksi bank menggunakan teknik machine learning, khususnya untuk tugas klasifikasi dan clustering. Dataset yang digunakan berisi data transaksi bank yang telah diedit dan disesuaikan untuk keperluan analisis.

## Struktur Project
- **[Clustering] Submission Akhir BMLP_valentino2008.ipynb**: Notebook untuk analisis clustering.
- **[Klasifikasi] Submission Akhir BMLP_valentino2008.ipynb**: Notebook untuk analisis klasifikasi.
- **bank_transactions_data_edited.csv / .xlsx**: Dataset utama yang digunakan.
- **data_clustering_with_target.csv / data_clustering.csv**: Dataset untuk eksperimen clustering.
- **model_clustering/**: Folder model clustering (jika ada).
- **.h5 files**: Model hasil training (Decision Tree, KNN, Logistic Regression, Random Forest, SVM, dan tuning).

## Fitur
- **Klasifikasi**: Membangun dan mengevaluasi model klasifikasi untuk memprediksi kategori transaksi.
- **Clustering**: Mengelompokkan data transaksi untuk menemukan pola tersembunyi.
- **Eksplorasi Model**: Eksperimen dengan berbagai algoritma machine learning.

## Cara Menjalankan
1. Pastikan sudah menginstall Python dan Jupyter Notebook.
2. Install library yang dibutuhkan:
   ```bash
   pip install pandas numpy scikit-learn matplotlib seaborn tensorflow
   ```
3. Buka notebook yang diinginkan menggunakan Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
4. Jalankan setiap cell secara berurutan.

## Requirements
- Python 3.8+
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
- tensorflow (untuk model .h5)
